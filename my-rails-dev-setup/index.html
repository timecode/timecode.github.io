My Rails Dev Setup

Here’s a list of everything I do to develop a rails app!

Prerequisites:

  1. I’ve already created a rails app. See creating-a-new-heroku-rails-4-app post.

  2. I’ve setup a BDD stack for the rails app. See setting-up-my-rails-bdd-stack post.

Then:

  1. Open browser windows with all necessary bookmarks loaded. See rails-dev-bookmarks post.

  2. Open a terminal window and:

    $ cd rails-app-name

  3. Open a duplicate terminal window with cmd-T, tear its tab off so it becomes its own window, move the window to a second monitor or somewhere where it’s visible at all times, then start a guard session:

    $ bundle exec guard

  4. Open the project in an editor:

    $ mate ./
          # OR
$ subl ./

  5. Add feature specifications (watch guard window show red failures).

  6. Implement features (watch guard window show green passes).

  7. Repeat steps 5 and 6. Note: Use the web browser windows setup in step 1 to verify the app, as well as for looking up docs etc
